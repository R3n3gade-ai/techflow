## Layout Characteristics to Preserve
- Strong top masthead with small top line and bold main title block.
- Split hero header:
  - left side = report title/date/context
  - right side = regime box with current regime and score
- Cream / parchment background blocks with dark navy main header block and gold accents.
- Clear numbered sections from 1 to 8.
- Mix of cards, summary metric boxes, and wide tables.
- Typography hierarchy: uppercase section names, bold orange/gold emphasis, dark body text.
- Dense but highly structured investor-briefing style.
- Page headers/footers repeat title, confidentiality, regime transition, score, and page number.
- Feels like an institutional PDF / partner monitor, not a web app dashboard.
